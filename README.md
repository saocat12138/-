# 融合超图学习的AD病人静息态fMRI动态功能连接分析

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

基于深度嵌入聚类（DEC）与超图学习的静息态 fMRI 动态功能连接分析框架，用于揭示阿尔茨海默病（AD）患者脑网络的高阶动态异常，为 AD 的神经病理机制解析和潜在生物标志物挖掘提供可靠的计算工具。

---

## 项目简介

本项目提出了一套融合**深度嵌入聚类（Deep Embedded Clustering, DEC）** 与**超图学习（Hypergraph Learning）** 的完整分析流程，突破了传统动态功能连接方法仅能表征脑区两两低阶关系的局限，实现了对多个脑区间高阶空间关系的精准刻画。

主要特点包括：
- **高维特征降维**：采用堆叠自编码器（SAE）对高维功能连接特征进行非线性降维与深度特征学习
- **动态微状态识别**：基于 DEC 对所有滑动窗口特征进行联合聚类，提取时变的稳定脑状态
- **高阶关系建模**：引入超图理论构建超边，刻画多脑区间的协同共激活模式
- **时空特性量化**：计算占空比（Duty Cycle）、平均持续时间（Mean Dwell Time）等动态指标
- **组间对比分析**：通过匈牙利算法实现状态对齐，支持健康对照与 AD 患者的统计差异检验

---

##  应用场景

- **健康人群脑动态分析**：揭示性别差异、认知功能相关的高阶网络组织模式
- **AD 病理机制研究**：识别 AD 患者在动态脑状态切换、网络整合能力等方面的异常
- **潜在生物标志物挖掘**：提取具有诊断价值的动态功能连接特征
